# moonrabbit
Very low resource terminal based file manager written in C.  
Aims to be posix compliant.
Keybindings are vim for the most part

## Dependencies  
    ncurses  
    
## Compile  
Before compiling edit the config... yeah  
    gcc -lncurses *.c -o moonrabbit
